## Premium Fruit Online Sales Website

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/39a2ed1961224f0cafde14039569fc06/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, merchants, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Announcements
5. Investment and Franchise
6. Contact Us
7. Website Introduction
8. Merchants
9. Fruits
10. Personal Center
11. Shopping Cart




**Merchant Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Announcement Management
6. Fruit Management
7. Fruit Review Management
8. Fruit Order Management



**Administrator Functions:**
1. Login
2. Personal Center
3. Announcement Type Management
4. Merchant Credit Type Management
5. First-level Category Management
6. Announcement Management
7. Fruit Management
8. Fruit Review Management
9. Fruit Collection Management
10. Fruit Order Management
11. Single Page Data Management
12. User Management
13. Merchant Management
14. Carousel Image Management
